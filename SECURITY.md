# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in `@fex-to/precise-compact`, please report it by emailing **gkhelloworld@gmail.com** or opening a private security advisory on GitHub.

**Please do not open public issues for security vulnerabilities.**

### What to expect:

- **Response time:** Within 48 hours
- **Updates:** Every 3-5 days until resolved
- **Disclosure:** After fix is released, we'll publish a security advisory

We take all security reports seriously and will work with you to address the issue promptly.
